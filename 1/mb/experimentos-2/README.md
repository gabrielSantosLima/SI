## Questão 1.1 (Latex)
```
\frac{1000}{100} = \frac{2}{x}\\
1000 * x = 200\\
x = \frac{200}{1000} = \frac{2}{10}\\
x = \text{0,2 reais}\\\\

\frac{1000}{100} = \frac{1.5}{y}\\
1000 * x = 150\\
y = \frac{150}{1000} = \frac{15}{100}\\
y = \text{0,15 reais}\\\\

x + y = 0.2 + 0.15 = \text{0.35 reais}
```

## Questão 1.2 (Latex)
```

\frac{1000}{800} = \frac{1.5}{y}\\
1000 * x = 1200\\
y = \frac{1200}{1000} = \frac{12}{10}\\

```
